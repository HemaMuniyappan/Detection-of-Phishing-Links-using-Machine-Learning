# Phishing URL Detection Using Machine Learning

A machine-learning-based web application that detects **phishing URLs** using
Random Forest classification. The system analyzes URL features to classify them
as **Legit** or **Fake**.

## Features

*Real-time phishing prediction  
* Machine learning (Random Forest)  
* URL feature extraction  
* Flask web interface  
* API-based prediction  
* Confusion matrix & charts  
* Training script included
